## Data Flow

1. **User вҶ’ Chat**: Primary interface. User describes needs in natural language.
2. **Chat вҶ’ Claude API**: Claude processes with tool-use, calls platform tools.
3. **Tools вҶ’ DB Agent**: All database operations go through the Database Agent.
4. **DB Agent вҶ’ PostgreSQL**: Permission-checked queries execute against the database.
5. **FileBrowser вҶ” Shared Volume**: File management through web UI.
